Fadeaway Hoops

FeelsLike

I always feel like I am mislead by reading the weather on my phone because it never includes the windchill and the feels like temperature. So I decided to do create an Android app that solves this problem. Users simply enter their location, temperature and windspeed and using the Yahoo Weather API the app displays the 'feels like' temperature. Some features I added recently include error-checking and storing the users location for future use.

  • Technologies: Java, Android Studio, SQL, Yahoo Weather API
